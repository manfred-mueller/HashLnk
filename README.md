# HashLnk
Compute and apply a hash for "WinX Menu" shortcuts.

## Usage
```
string myWinXLnk = @"C:\Users\Xmoer\AppData\Local\Microsoft\Windows\WinX\Group4\Paint.lnk";
new HashLnk(myWinXLnk);
```
